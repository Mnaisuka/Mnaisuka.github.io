如何在《The Long Dark》中安装Mod

当前Mod仅支持 Steam 版本的游戏，并且仅适用于 Windows 系统。

本网站与 Hinterland Games 无任何关联，仅用于提供Mod下载入口。

  1. 如果你没有关闭崩溃报告和数据分析，请先启动游戏并关闭它们。
    这很重要！ 除非确定是官方问题，否则不要打扰开发者。
  2. 在安装Mod前务必备份存档！虽然Mod通常经过测试，但仍可能出现问题。
    存档路径：C:\Users\[用户名]\AppData\Local\Hinterland\TheLongDark
  3. 首先，请确保安装了 Melon Loader 运行环境：
    • Microsoft Visual C++ 2015-2019 运行库（64位）
    • .NET 6.0 Runtime
  4. 接下来安装 MelonLoader：
    • 下载 MelonLoader.Installer.exe
    • 运行安装程序（位置不限）
    • 选择 TheLongDark（或手动添加）
    • 选择版本 0.7.2
      （勾选“Enable Nightly Builds”并选择最新版本）
    • 等待安装完成
  5. 下载Mod（如为zip需解压），将 .dll / .modcomponent 文件放入游戏目录中的 Mods 文件夹
    （注意查看Mod说明，有些包含多个版本）
  6. 确保安装所有依赖项
  7. 启动游戏，享受Mod吧 :)

常见问题

出现 “iL2CPP-init detour failed” 错误

需要安装 .NET 6 Runtime

提示缺少 UnhollowerBaseLib

该Mod未适配当前版本，请删除该Mod

游戏启动崩溃

请确认使用 MelonLoader 0.7.2

Epic版本无法运行Mod

Epic版本有防篡改系统，目前无法支持Mod

如何卸载 MelonLoader

重新运行安装程序并选择游戏，即可卸载

或手动删除以下文件夹：MelonLoader、Mods、Plugins、UserData、UserLibs 和 version.dll

Mod支持剧情模式吗？

目前不支持。生存模式与剧情模式已分离，理论上可以分别安装，但存在加载问题。
